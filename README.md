# Basic Sales Data Analysis in Python

This project demonstrates basic data analysis skills using Python, Pandas, NumPy and Matplotlib.
It works with a small synthetic sales dataset and shows how to:

- Load data from CSV
- Clean and enrich data (add derived columns)
- Compute summary statistics
- Aggregate metrics by region and product
- Create simple visualisations
